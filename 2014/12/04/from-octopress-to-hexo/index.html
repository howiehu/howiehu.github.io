从Octopress到Hexo

经过一番折腾，将博客程序从基于 Ruby 的 Octopress 切换至了最近很火的，基于 JavaScript 的 Hexo。

切换后的感受：

  1. 毕竟是基于 JavaScript 的，整个浏览体验更快捷；
  2. JavaScript 生命力旺盛，做出来的东西确实轻量且易于维护。

推荐其他仍然在用 Octopress 的朋友也尝试下 Hexo，这里有一篇从其他博客程序迁移到 Hexo 的官方指南：Migration。
